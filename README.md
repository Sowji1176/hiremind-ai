# HireMind AI

HireMind AI is an AI-powered recruitment dashboard that analyzes resumes using an ATS-based scoring system and provides accurate insights for better hiring decisions.

## 🚀 Features
- AI-powered resume analysis
- ATS-based scoring system
- Accurate candidate data extraction (no fake data)
- Dashboard for candidate management
- Secure authentication system

## 🛠️ Tech Stack
- Frontend: React + Tailwind CSS
- Backend: FastAPI
- Database: MongoDB
- AI: OpenAI
